# ✏️ SketchSync

A real-time collaborative drawing application inspired by Excalidraw. SketchSync allows multiple users to draw together on a shared canvas with instant updates powered by WebSockets.
## 🚀 Features

* 🎨 Real-time collaborative drawing
* 🔗 Multi-user support on shared canvas
* ⚡ Instant updates using WebSockets
* 🧠 Efficient data synchronization
* 🛠️ Minimal and scalable backend design

## 🏗️ Tech Stack

### Backend
* Node.js
* Express
* WebSocket (ws)
* TypeScript

### Frontend
* Next.js
* Canvas API
